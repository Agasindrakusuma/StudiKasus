HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat dan menyusun konten yang dapat ditampilkan di web browser. HTML berfungsi untuk mengatur struktur dan tampilan halaman web, termasuk teks, gambar, tautan, tabel, formulir, video, dan elemen-elemen lainnya. HTML menggunakan tag-tag yang ditempatkan di antara tanda "<" dan ">" . Tag-tag ini memberi instruksi kepada browser tentang bagaimana memformat dan menampilkan konten di halaman web. Beberapa tag HTML umum meliputi:

  • Menandakan awal dan akhir dokumen HTML.
  • Berisi informasi-informasi tentang dokumen seperti judul, meta data, dan script.
  • Berisi konten yang akan ditampilkan di halaman web, seperti teks, gambar, tautan, dan elemen-elemen lainnya.
  • Tag-tag untuk judul dengan tingkat kepentingan yang berbeda.

  • Tag untuk membuat tautan atau hyperlink.
  • Tag untuk menyisipkan gambar.
  • Tag untuk membuat tabel.
  • Tag untuk membuat formulir interaktif.
  • Tag untuk mengelompokkan dan mengatur elemen-elemen dalam blok.
  • Tag untuk mengelompokkan dan mengatur elemen-elemen dalam baris.

Misalnya, atribut "src" digunakan dalam tag `` untuk menentukan lokasi gambar yang akan ditampilkan. HTML bekerja bersama dengan CSS (Cascading Style Sheets) dan JavaScript untuk memberikan tampilan dan interaktivitas yang lebih kaya pada halaman web. CSS digunakan untuk mengatur tampilan elemen-elemen HTML, sedangkan JavaScript digunakan untuk menambahkan logika dan interaktivitas ke halaman web.



CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengontrol tampilan dan presentasi halaman web yang ditulis dalam HTML atau markup language lainnya. Dengan menggunakan CSS, Anda dapat mengubah penampilan visual elemen-elemen HTML, seperti warna, font, ukuran, tata letak, dan banyak lagi. Tujuan utama CSS adalah memisahkan presentasi atau tampilan dari struktur atau konten halaman web. Ini berarti Anda dapat membuat perubahan tampilan yang konsisten di seluruh situs web hanya dengan mengubah file CSS terkait, tanpa harus memodifikasi setiap halaman individu. Pendekatan ini memungkinkan pemeliharaan yang lebih efisien dan fleksibilitas dalam mengelola desain situs web.

  • CSS bekerja dengan prinsip "cascade" atau "hirarki". Ketika browser membaca halaman web, ia menerapkan aturan CSS sesuai dengan urutan yang ditentukan. Jika ada beberapa aturan yang diterapkan pada elemen yang sama, maka aturan yang memiliki spesifisitas yang lebih tinggi akan diikuti. Misalnya, aturan yang dideklarasikan secara langsung pada elemen menggunakan atribut `style` akan memiliki spesifisitas yang lebih tinggi daripada aturan yang dideklarasikan dalam file CSS eksternal.
  • CSS mendukung berbagai jenis selektor untuk memilih elemen atau kelompok elemen tertentu. Misalnya, Anda dapat memilih elemen berdasarkan tipe, kelas, ID, atribut, hierarki, dan lainnya. Selain itu, CSS juga menyediakan berbagai properti dan nilai untuk mengatur berbagai aspek tampilan, seperti warna, jenis font, tata letak, margin, padding, latar belakang, transisi, animasi, dan banyak lagi.
  • CSS juga memiliki fitur-fitur lanjutan, seperti penggunaan media query untuk mengatur tampilan yang responsif terhadap perangkat, fleksbox dan grid untuk mengatur tata letak halaman yang lebih kompleks, dan dukungan untuk transformasi, filter, dan efek visual lainnya.
    • Dengan menggunakan CSS bersamaan dengan HTML dan seringkali JavaScript, Anda dapat menciptakan tampilan yang menarik, konsisten, dan interaktif pada halaman web, serta memberikan pengalaman pengguna yang lebih baik.
